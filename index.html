Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le web, ou World Wide Web (WWW), est un système mondial qui permet d'accéder à des informations et contenus variés comme des textes, des images ou des vidéos à travers des navigateurs. Il relie des pages et des ressources via des liens hypertextes. Ce réseau est essentiel pour la communication, l'apprentissage et de nombreuses activités en ligne.

Internet

L’internet est un réseau mondial qui connecte des millions d’ordinateurs et d’appareils, permettant l’échange d’informations. Il est utilisé pour communiquer, accéder à des ressources, et partager des contenus. Il constitue la base technologique du web et de nombreux services numériques.

Cloud

Le cloud désigne un ensemble de serveurs accessibles via internet pour stocker, gérer et traiter des données à distance. Il permet aux utilisateurs d’accéder à leurs fichiers et applications depuis n’importe quel appareil connecté. Cette technologie offre flexibilité, évolutivité et réduction des coûts matériels.

W3C

Le W3C (World Wide Web Consortium) est une organisation internationale qui élabore des standards pour garantir l’accessibilité, l’interopérabilité et l’évolution du web. Fondé en 1994, il regroupe des experts et des entreprises du monde entier. Ses recommandations définissent les bases techniques du web moderne.

Partie 2 - Distinguez les langages du Web

Langages client

HTML (HyperText Markup Language)
CSS (Cascading Style Sheets)
JavaScript

Navigateurs

Un navigateur (ou browser) est un logiciel qui permet d'accéder et de visualiser des pages web sur internet. Il interprète les fichiers HTML, CSS et JavaScript pour afficher correctement le contenu d'un site.
Exemples de navigateurs :
.Google Chrome
.Mozilla Firefox
.Safari
.Microsoft Edge

Langages serveur

Les langages côté serveur (ou serveur-side) sont utilisés pour traiter les requêtes des utilisateurs, gérer les bases de données et générer dynamiquement du contenu avant de l'envoyer au client (navigateur).
Voici quelques exemples courants :
.PHP
.Python
.Ruby
.Ruby,
.Node.js
.Java
.C#

Frameworks

Un framework est un ensemble d'outils et de bibliothèques préconfigurées qui facilite le développement d'applications en fournissant une structure de base et des solutions pour des tâches courantes. Il permet aux développeurs de gagner du temps en évitant de réinventer des fonctionnalités déjà existantes et en garantissant la cohérence et la maintenabilité du code.

Bases de données

Une base de données est un système organisé pour stocker, gérer et récupérer des informations de manière efficace.

Site responsive

Un site responsif est un site web conçu pour s'adapter automatiquement à différentes tailles d'écran, offrant ainsi une expérience optimale sur ordinateurs, tablettes et smartphones.

Application mobile

Une application mobile est un logiciel conçu pour être utilisé sur des appareils mobiles comme les smartphones et tablettes, offrant des fonctionnalités spécifiques à l'utilisateur.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est un ordinateur ou un système qui fournit des services, des ressources ou des données à d'autres ordinateurs (clients) via un réseau.

Adresse IP

Une adresse IP (Internet Protocol) est une série de chiffres unique attribuée à chaque appareil connecté à un réseau, permettant son identification et sa communication avec d'autres appareils.

Nom d’hôte et DNS

Le nom d'hôte est un nom unique attribué à un appareil sur un réseau pour faciliter son identification. Le DNS (Domain Name System) est un système qui traduit les noms d'hôtes en adresses IP pour permettre la communication entre les appareils sur Internet.

Protocoles

Les protocoles sont des règles et des conventions qui régissent la communication entre les appareils sur un réseau, garantissant l'échange de données de manière standardisée et fiable (ex. HTTP, FTP, TCP/IP).

Partie 4 - Identifiez les métiers du web

Développeurs

Les développeurs créent et maintiennent les applications web ou mobiles en écrivant du code, généralement avec des langages comme HTML, CSS, JavaScript, ou d'autres langages côté serveur.

Product Managers et chefs de projets

Le Product Manager définit la vision, les fonctionnalités et les priorités d'un produit digital, tandis que le Chef de Projet gère la planification, les ressources et la mise en œuvre des projets web, en veillant à respecter les délais et le budget.

Designers

Les designers sont responsables de l'apparence et de l'ergonomie d'un site ou d'une application, créant des interfaces visuelles attrayantes et intuitives pour les utilisateurs.

Testeurs

Les testeurs (ou QA - Quality Assurance) vérifient la qualité et le bon fonctionnement des produits web en effectuant des tests pour détecter des bugs, des erreurs ou des améliorations possibles.

Techniciens informatique

Les techniciens informatiques assurent la maintenance, le support et la gestion des systèmes informatiques, des serveurs et des infrastructures techniques nécessaires au bon fonctionnement des applications web et des réseaux.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

L'habillage du site Web. Il définit l'apparence visuelle du contenu, comme les couleurs, les polices, les marges et la disposition des éléments.

JavaScript

Front-End

L'interactivité du site Web. Il permet d'ajouter des fonctionnalités dynamiques, comme des animations, des interactions avec l'utilisateur ou des mises à jour en temps réel sans recharger la page.

JavaScript

Back-End

JavaScript pour le back-end est un langage de programmation utilisé sur les serveurs grâce à Node.js, permettant de gérer des bases de données, des requêtes utilisateurs et de créer des API pour des applications web.

PHP

Back-End

Un langage côté serveur qui permet de développer des sites web interactifs, de gérer des bases de données et de générer des contenus adaptés à chaque utilisateur.

Java

Back-End

Un langage de programmation polyvalent utilisé pour créer des applications web robustes, des services backend et des systèmes complexes nécessitant une grande fiabilité.

Python

Back-End

Un langage de programmation accessible et puissant, utilisé pour le développement web, l’analyse de données, l’intelligence artificielle et la gestion de bases de données.

C#

Back-End

Un langage de programmation orienté objet, souvent utilisé pour le développement d’applications web avec le framework .NET, ainsi que pour des logiciels et des jeux.

SQL

Bases de données

Un langage de requête conçu pour gérer et interroger des bases de données relationnelles, permettant de stocker, récupérer et manipuler des données efficacement.

Boîte à outils du développeur

  • IDE comme VS Code :
    Un IDE (Environnement de Développement Intégré) est un logiciel qui fournit des outils pour coder, tester et déboguer facilement, comme l’éditeur de texte, l’autocomplétion et le gestionnaire de projets.
  • Github :
    Une plateforme en ligne qui permet de stocker, gérer et partager du code source, tout en facilitant la collaboration entre développeurs grâce au contrôle de version avec Git.
  • Librairies / bibliothèques :
    Les bibliothèques sont des collections de fonctions ou de classes préécrites qui simplifient et accélèrent le développement en offrant des solutions prêtes pour des tâches courantes.
  • Frameworks :
    Un framework est un ensemble d'outils et de bibliothèques qui facilitent le développement d'applications en fournissant une structure de base et des fonctionnalités prêtes à l'emploi.
    Exemples de frameworks :
    .React – Un framework JavaScript pour construire des interfaces utilisateurs dynamiques.
    .Django – Un framework Python pour développer des applications web rapides et sécurisées.
    .Ruby on Rails – Un framework Ruby pour créer des applications web robustes et évolutives.
